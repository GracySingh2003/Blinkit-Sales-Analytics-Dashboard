## 📊 Project Title:
Blinkit Sales Analytics Dashboard – Retail Performance Insights (Power BI)

## 📝 Project Overview: 
This project showcases an **interactive retail analytics dashboard** built using Power BI for Blinkit (India’s last-minute delivery app).  
It provides deep insights into sales performance, product categories, outlet types, and customer behavior to support data-driven decision-making.

## 🎯 Purpose:  
- Analyze retail sales performance  
- Identify top-performing products and categories  
- Evaluate outlet performance across locations and sizes  
- Understand customer preferences and ratings  
- Improve inventory and business strategies

## 🛠️ Tech Stack:  
- 📊 **Power BI Desktop** – Dashboard development  
- 🔄 **Power Query** – Data cleaning & transformation  
- 📐 **DAX** – Measures & KPI calculations  
- 📁 **Excel / CSV** – Data source

## 📂 Data Source:  
- Retail dataset including item types, outlet size, location, sales, and ratings  
- Built using a **YouTube Power BI tutorial (project-based learning)**

## 🚀 Features & Highlights:  

### 🔍 Business Problem: 
Retail businesses struggle to analyze large datasets to identify high-performing products, optimize inventory, and understand customer preferences across multiple outlet types and regions.

### 🎯 Goal:  
To build an **interactive dashboard** that provides actionable insights into retail performance, helping businesses improve sales strategies and operational efficiency.

### 📈 Key Metrics:  
- 💰 Total Sales: **$1.20M**  
- 📦 Number of Items: **8523**  
- 💵 Average Sales: **$141**  
- ⭐ Average Rating: **3.9**

### 📊 Key Visuals:  
- 📌 KPI Cards (Sales, Avg Sales, Items, Rating)  
- 🍩 Fat Content Analysis (Low Fat vs Regular)  
- 📊 Item Type Sales (Category-wise performance)  
- 📉 Outlet Establishment Trend (Year-wise analysis)  
- 🍩 Outlet Size Distribution  
- 📊 Outlet Location Analysis (Tier 1, 2, 3)  
- 📋 Outlet Type Table (Supermarket & Grocery comparison)  
- 🎛️ Filter Panel (Outlet Size, Location, Item Type)

### ⭐ Key Features:  
- Interactive filter panel for dynamic insights  
- Multi-dimensional analysis (product, outlet, location)  
- Identification of high-performing categories  
- Comparative analysis across outlet types  
- Clean and business-focused dashboard design

## 💡 Key Insights:
-  Tier 3 outlets generate the highest sales  
-  Regular fat products dominate sales contribution  
-  Fruits, snacks, and household items perform best  
-  Supermarket Type 1 leads in total sales  
-  Sales trends vary based on outlet establishment year
